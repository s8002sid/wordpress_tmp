== Installation ==

This section describes how to install the plugin and get it working.

e.g:

1. Upload the entire 'vina-camera-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'Vina Camera Slideshow Widget' to your sidebar
